Análisis de Funcionalidades:
1.- El sistema cuenta con una barra de navegación que redirecciona a distintas secciones del sitio web.
2.- Se implementó un formulario de registro de usuario e inicio de sesión.
3.- Cartas gitarorias que entregan información sobre la importancia de la vida submarina cuando el usuario se posiciona encima de uno de ellos.
4.- Acordeon con información más detallada de las metas cuando el usuario presiona una de ellas.
5.- Carrusel con las agrupaciones, imágenes (usuario ingresado)
6.- Sección para visualizar los comentarios de los usuarios registrados
7.- Reportajes exclusivos para usuarios registrados que permite visualizar/descargar pdf's
8.- Sección para usuarios registrados que pueden crear comentarios y puedan ser visualizados en la página principal
9.- Link's de youtube para visualizar videos y documentales relacionados con la ODS 14.
10.- Contiene una barra de búsqueda que facilita encontrar recursos que se encuentren en el sitio web.
